# TC Coolers V3.1 Stable Prototype

This version removes the inaccurate procedural 3D cap and uses the supplied CTC cap reference image directly.

## Why this version is stable
- The TC Coolers logo is embedded directly into index.html.
- The cap reference image is embedded directly into index.html.
- No image paths are required, so GitHub Pages cannot return image 404 errors.

## Important
This is a 2D visual customiser, not a genuine 360° model. A true, accurate 360° version requires a professionally created 3D model or scan of the Country Trucker Caps deep-fit/high-peak cap.

## Upload
Replace the existing files in your GitHub repository with:
- index.html
- style.css
- script.js
- README.md

Then commit changes and hard-refresh the live page with Ctrl+F5.
